# Custom Types

Prismic Adapter supports handling different types of page syncing on the Shopify based on the types defined under the hood on the backend application.

# Currently supported custom types

  • Pages - Pages custom type sync data on the Shopify pages
  • Blogs - Blogs custom type sync data on the Shopify blog pages
  • Articles - Articles custom type sync data on the Shopify article pages

# Expected custom types in future

  • Collections - Collections custom type sync data on the Shopify specific collections pages
  • Products - Products custom type sync data on the Shopify specific products pages
  • Global - Global custom type sync data on the Shopify global meta-fields
