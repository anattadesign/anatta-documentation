    # App And Prismic Setup

    This steps includes creating custom app into the shopify partner account and then installing it into the store as well setting up on prismic side and creating webhooks to sync data.

    WARNING

    Make sure that you have permission to create app on shopify partner account.

    # Follow the following steps to get started:

    Step 1
    Create a custom shopify app:

    Learn how Create a custom app on shopify

    Step 2
    Install the custom app:

    Learn how to install the custom app

    Step 3
    Configure prismic repo to sync with app :
    For our custom app to sync data between prismic repo and shopify entity, we need configure on prismic side.

    Learn how to configure primsic repo to sync with app

    Step 4
    Configure app to sync with prismic repo :
    Once prismic repo has been setup , we need to configure our app with the required credential and preview url.

    Learn how to configure custom app

    Step 5
    Setup Preview :
    Setup the preview, that'll allow to preview draft data without actually publishing them from prismic

    Learn how to configure prismic preview
